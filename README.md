# GoCash

Hekaton projekt – demo za **Flik 2.0**.

Ta demo vključuje stare načine pošiljanja in zahtevnja denarja, obogatene z novimi funkcionalnostmi:

- Razdelitev plačila med več ljudmi  
- Ustvarjanje skupin za deljenje plačil  
- Deljenje zahtev za plačilo z QR kodo
